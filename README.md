# 🚀 UX AI Prompt Library

A collection of engineered prompts and skills to automate UI/UX research and design ops at **Wild Me**.

### 📊 The "Efficiency" Win
* **Research Prep:** 60 min → 3 min per participant.
* **Accuracy:** 100% verification via mandatory source links.

> [!TIP]
> **How to use:** Open the prompt link, click the **Copy icon** on the code block, and paste it into Gemini, GPT-4, or Claude.
