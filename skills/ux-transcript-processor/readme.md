## How to use (Claude Desktop Skills)

### One-time setup
1. Open Claude Desktop
2. Go to Sidebar → Customize → Skills
3. Upload `ux-transcript-processor.skill.md`

### Usage
1. Open new chat
2. Upload Fathom transcript (.txt)
3. Copy and Paste the following:
```
   Use the UX Transcript Processor skill to process this transcript.
```
4. Run
